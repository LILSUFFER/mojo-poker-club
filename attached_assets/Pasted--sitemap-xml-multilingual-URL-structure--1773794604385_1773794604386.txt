Нужно исправить sitemap.xml под новую multilingual URL structure.

Важно:
языки больше НЕ используют query params like ?lang=ru.
Теперь языковые версии работают через path prefixes:

/           -> English default
/ru/        -> Russian
/de/        -> German
/es/        -> Spanish
/fr/        -> French
/it/        -> Italian
/pt/        -> Portuguese
/tr/        -> Turkish
/kk/        -> Kazakh
/uz/        -> Uzbek
/zh/        -> Chinese
/ja/        -> Japanese

Сделай sitemap.xml строго так:

1. Используй XML namespaces:
   xmlns="http://www.sitemaps.org/schemas/sitemap/0.9"
   xmlns:xhtml="http://www.w3.org/1999/xhtml"

2. Для каждой canonical page должна быть одна запись <url>, и внутри:
   - <loc>
   - <lastmod>
   - <changefreq>
   - <priority>
   - <xhtml:link rel="alternate" hreflang="..."> для всех языков
   - <xhtml:link rel="alternate" hreflang="x-default" ...>

3. Очень важно:
   - НЕ использовать ?lang=
   - НЕ делать отдельные <url> записи для каждой языковой версии
   - у canonical page должна быть одна <url> запись
   - alternate language versions должны быть указаны через xhtml:link

4. Пример для homepage:

<?xml version="1.0" encoding="UTF-8"?>
<urlset
  xmlns="http://www.sitemaps.org/schemas/sitemap/0.9"
  xmlns:xhtml="http://www.w3.org/1999/xhtml">

  <url>
    <loc>https://mojopokerclub.com/</loc>
    <lastmod>2026-03-18</lastmod>
    <changefreq>weekly</changefreq>
    <priority>1.0</priority>

    <xhtml:link rel="alternate" hreflang="en" href="https://mojopokerclub.com/" />
    <xhtml:link rel="alternate" hreflang="ru" href="https://mojopokerclub.com/ru/" />
    <xhtml:link rel="alternate" hreflang="de" href="https://mojopokerclub.com/de/" />
    <xhtml:link rel="alternate" hreflang="es" href="https://mojopokerclub.com/es/" />
    <xhtml:link rel="alternate" hreflang="fr" href="https://mojopokerclub.com/fr/" />
    <xhtml:link rel="alternate" hreflang="it" href="https://mojopokerclub.com/it/" />
    <xhtml:link rel="alternate" hreflang="pt" href="https://mojopokerclub.com/pt/" />
    <xhtml:link rel="alternate" hreflang="tr" href="https://mojopokerclub.com/tr/" />
    <xhtml:link rel="alternate" hreflang="kk" href="https://mojopokerclub.com/kk/" />
    <xhtml:link rel="alternate" hreflang="uz" href="https://mojopokerclub.com/uz/" />
    <xhtml:link rel="alternate" hreflang="zh" href="https://mojopokerclub.com/zh/" />
    <xhtml:link rel="alternate" hreflang="ja" href="https://mojopokerclub.com/ja/" />
    <xhtml:link rel="alternate" hreflang="x-default" href="https://mojopokerclub.com/" />
  </url>

</urlset>

5. Используй priority так:
   - homepage `/` -> 1.0
   - main SEO pages like `/clubs/massiv`, `/clubs/mojo`, `/games` -> 0.8
   - secondary pages like `/about` -> 0.5
   - technical pages like `/join`, `/create-account`, `/download`, `/install` -> exclude from sitemap completely

6. Не включай в sitemap:
   - /join
   - /create-account
   - /download
   - /install
   - auth pages
   - technical pages
   - duplicate URLs
   - query parameter URLs

7. В sitemap должны быть только canonical SEO pages:
   - https://mojopokerclub.com/
   - https://mojopokerclub.com/clubs/massiv
   - https://mojopokerclub.com/clubs/mojo
   - https://mojopokerclub.com/games
   - https://mojopokerclub.com/about (if exists)

8. Для каждой canonical page добавить hreflang alternate URLs на все 12 языков через path prefixes.
Пример для a page:
   canonical:
   https://mojopokerclub.com/games

   alternates:
   https://mojopokerclub.com/games
   https://mojopokerclub.com/ru/games
   https://mojopokerclub.com/de/games
   https://mojopokerclub.com/es/games
   ...

9. Не генерируй sitemap автоматически из всех routes.
   Используй explicit whitelist страниц.

10. Response Content-Type:
   application/xml; charset=utf-8

11. Верни valid raw XML string, not plain text, not JSON, not array.
